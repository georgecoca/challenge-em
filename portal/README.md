# Setup
Before running the following commands make sure you are not already running other docker containers or having other services that blocks the following ports:
- 3000 (vite server)

### Copy local env file
```bash
cp .env.example .env
```

### Run the container

```bash
docker-compose up -d
```

# Access
The Portal is running at this web address:
```
http://localhost:3000
```
